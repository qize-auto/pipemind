# 🔧 PipeMind — AI 工作流引擎

> 可视化拖拽构建 AI 工作流，像拼乐高一样串联搜索、LLM、输出等节点。

## 快速开始

```bash
# 1. 启动后端
cd backend && npm start
# → http://localhost:3001

# 2. 启动前端（新终端）
cd frontend && npm run dev
# → http://localhost:5173
```

## 使用

1. 从左侧节点库**拖拽**节点到画布
2. **连接**节点（从节点右侧拖线到下一个节点左侧）
3. 双击节点编辑参数（搜索关键词、LLM 提示词等）
4. 点击右上角 **▶ 运行**

## 节点类型

| 节点 | 功能 | 图标 |
|------|------|------|
| **搜索** | 联网搜索信息（支持时效过滤） | 🔍 |
| **LLM 处理** | AI 文本处理（支持自定义提示词和模型） | 🧠 |
| **输出** | 格式化展示结果（文本/Markdown/JSON） | 📤 |

## 技术栈

- **前端**: React 18 + React Flow + Tailwind CSS v4 + Vite
- **后端**: Node.js + Express
- **架构**: DAG 工作流引擎，支持异步拓扑排序执行
